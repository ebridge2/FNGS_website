register()¶

A function to perform self registration followed by template registration.

self_align()¶

A function to perform self alignment. Uses a local optimisation cost function to get the two images close, and then uses bbr to obtain a good alignment of brain boundaries.

template_align()¶

A function to perform template alignment. First tries nonlinear registration, and if that does not work effectively, does a linear registration instead. NOTE: for this to work, must first have called self-align.
