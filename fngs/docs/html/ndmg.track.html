ndmg.track.track module¶

class ndmg.track.track.track¶
eudx_basic(dwi_file, mask_file, gtab, stop_val=0.1)¶

Tracking with basic tensors and basic eudx - experimental We now force seeding at every voxel in the provided mask for simplicity. Future functionality will extend these options. Positional Arguments:

dwi_file:
  • File (registered) to use for tensor/fiber tracking
mask_file:
  • Brain mask to keep tensors inside the brain
gtab:
  • dipy formatted bval/bvec Structure
Optional Arguments:
stop_val:
  • Value to cutoff fiber track
